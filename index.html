About This Project

This generative art system uses Total Serialism's algorithmic composition techniques to create Fidenza-inspired flowing artworks. Mathematical patterns from music (Euclidean rhythms, harmonic progressions) drive visual elements.

Each refresh generates a completely unique A3-sized artwork (297×420mm @ 150 DPI).
