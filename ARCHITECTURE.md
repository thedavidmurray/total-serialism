## Design Patterns

### 1. **Centralized Parameters Pattern**
```javascript
const params = {
  param1: defaultValue1,
  param2: defaultValue2,
  // All algorithm configuration in one object
};
```

### 2. **Callback Pattern (PresetManager)**
```javascript
new PresetManager({
  algorithmId: 'unique-id',
  container: '#preset-container',
  onSave: () => params,
  onLoad: (preset) => { /* restore state */ },
  onRandomize: () => { /* randomize params */ }
});
```

### 3. **Utility Class Pattern**
```javascript
// Static utility methods
UIUtils.updateUIFromParams(params);
ExportUtils.exportSVG(paths, width, height, filename);
```

### 4. **Event-Driven UI Sync**
```javascript
UIUtils.setupParamSync(params, (key, value, allParams) => {
  // Automatic callback on any parameter change
  regenerate();
});
```
