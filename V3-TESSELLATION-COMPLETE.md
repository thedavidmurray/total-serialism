### Voronoi
- **Best for**: Organic, natural patterns
- **Looks like**: Cells, bubbles, territories
- **Complexity**: Controls cell gradient
- **Density**: Number of seed points

### Delaunay
- **Best for**: Crystalline, sharp structures
- **Looks like**: Shattered glass, gems
- **Complexity**: Triangle connectivity
- **Density**: Point distribution

### Penrose
- **Best for**: Mathematical beauty
- **Looks like**: Quasicrystals, tile work
- **Complexity**: Subdivision levels
- **Chaos**: Position wobble

### Islamic

- **Looks like**: Traditional geometric art
- **Complexity**: Layer details
- **Scale**: Grid size

### Waves
- **Best for**: Fluid, dynamic patterns
- **Looks like**: Water ripples, sound waves
- **Complexity**: Frequency range
- **Density**: Number of sources
