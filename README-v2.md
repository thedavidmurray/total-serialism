1. Open `total-serialism-fidenza-v2.html` in your browser

4. Save as PNG (`P`) or SVG (`S`)

## Keyboard Shortcuts

- `R` - Regenerate

- `S` - Save SVG

## Technical Details

### Algorithmic Techniques
- **Euclidean Rhythms**: Mathematical patterns for spatial distribution
- **Flow Fields**: Multi-layer sine/cosine wave combinations
- **Drunk Walks**: Organic turbulence generation
- **Fibonacci Sequences**: Natural proportions and counts
- **Golden Ratio**: Spiral and phyllotaxis patterns

### Performance Notes
- Lower field resolution = better performance
- Fewer particles = smoother animation
- SVG export captures current frame state

### File Sizes
- PNG: ~2-5MB (high resolution)
- SVG: ~1-10MB (depends on complexity)
- Presets: ~1KB (JSON parameters)

## Tips for Best Results

1. **Start with presets**: Use "Randomize All" then fine-tune
2. **Layer complexity**: Combine high flow complexity with low turbulence
3. **Color harmony**: Lower saturation often looks more sophisticated
4. **Rhythm patterns**: Prime numbers create interesting distributions
5. **Mixed modes**: Try different render modes with same parameters

## Creative Suggestions

- **Minimalist**: Low particle count, single color scheme, no turbulence
- **Chaotic**: High turbulence, mixed render mode, complex flow
- **Organic**: Wave behaviors, analogous colors, medium density
- **Geometric**: Grid spawn, low life range, dot field mode
- **Classic Fidenza**: Flow lines, complementary colors, golden spiral
